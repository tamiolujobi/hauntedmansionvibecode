Tami's Haunted Mansion

How to open locally:

- Open `index.html` in a browser (double-click or `File -> Open` in browser).
- Pages: `index.html` (Living Room), `bedroom.html` (Bedroom), `kitchen.html` (Kitchen).

Notes:
- Hover (or tap) your name on the landing page to spawn ghosts and bats.
- Click dolls or foods to trigger small animations.
- All assets are inline/SVG and the site is mobile responsive. Edit the HTML to change text, foods, or dolls.
